# GarmentIQ Web Portal Prototype

This folder contains a VS Code-friendly front-end prototype for the GarmentIQ role-based portal.

## What is included

- Improved dashboard UI for `Manager`, `Staff`, and `Analyst`
- Cleaner and more professional module language
- A sample module preview table inspired by your supplier screen
- Responsive layout so the design still works on smaller screens

## How to run

Because this version is plain HTML, CSS, and JavaScript, you can open it directly in VS Code and run it with any simple local server.

Examples:

- VS Code Live Server extension
- `python3 -m http.server` inside this folder

Then open `index.html`.
